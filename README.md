# ChatSupport UI

A custom React UI component library for ChatSupport, built with TypeScript and Storybook.

## Installation

```bash
npm install chatsupport-ui
```

## Usage

```typescript
import { /* components */ } from 'chatsupport-ui';
```

## Development

### Install Dependencies
```bash
npm install
```

### Development Mode
Watch mode for automatic compilation:
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Storybook Documentation
```bash
npm run storybook
```

Build Storybook for static hosting:
```bash
npm run storybook:build
```

### Testing
```bash
npm test
```

Watch mode:
```bash
npm run test:watch
```

### Type Checking
```bash
npm run lint
```

## Project Structure

- **src/** - Source code
  - **components/** - React components
  - **hooks/** - Custom React hooks
  - **types/** - TypeScript type definitions
  - **index.ts** - Main entry point
- **dist/** - Compiled distribution files
- **tests/** - Unit and integration tests
- **stories/** - Storybook stories for component documentation
- **.storybook/** - Storybook configuration

## Technologies

- **React 19** - UI framework
- **TypeScript 5** - Type safety
- **Jest** - Testing framework
- **React Testing Library** - Component testing utilities
- **Storybook 10** - Component documentation and development
- **Babel** - JavaScript transpiler
